## Enabling AppImage Support in Ubuntu
```sh
sudo apt update && sudo apt upgrade
sudo apt install libfuse2
```
## Installing Git
```sh 
sudo apt install git
git config --global user.email "you@example.com"
git config --global user.name "Your Name"

```
